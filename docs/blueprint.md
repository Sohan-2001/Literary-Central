# **App Name**: Literary Central

## Core Features:

- Book Management: Enable CRUD operations for books, including filtering and sorting.
- Author Management: Enable CRUD operations for authors.
- User Management: Enable the creation and listing of users.
- Book Borrowing: Allow marking a book as borrowed by a user.
- Book Return: Allow marking a borrowed book as returned.
- User Borrowing History: List all borrowed books for a specific user.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke a sense of trust and authority.
- Background color: Light blue-gray (#ECEFF1).
- Accent color: Soft orange (#FFAB40) to add a touch of warmth and highlight key actions.
- Body font: 'Inter', sans-serif, for clean and modern readability.
- Headline font: 'Space Grotesk', sans-serif, for clear and modern readability.
- Use minimalist, line-style icons to represent entities (books, authors, users) and actions (borrow, return, edit).
- Employ a grid-based layout for consistent spacing and alignment.
- Use subtle transitions when navigating between views or performing actions.
